# ✨ Vision

MoodIT est une plateforme universitaire moderne visant à améliorer l’expérience académique à l’Université de Sherbrooke.

Le projet combine :
- 💬 des espaces collaboratifs pour les cours,
- 🧠 des outils de révision intelligents,
- ⚡ des compétitions et défis interactifs,
- 🤖 un agent IA de rétroaction professorale,
- 📚 une communauté centralisée pour les étudiants.

---

# 🧩 Fonctionnalités

## 🎓 Académique

### 💬 Forums de cours
Des espaces organisés par :
- cours,
- session,
- cohorte,
- équipe de projet.

Avec :
- réponses imbriquées,
- votes,
- tags,
- recherche rapide,
- partage de ressources.

---

### 🤖 Agent MCP de rétroaction professorale
Un système structuré permettant aux étudiants d’envoyer :
- ✅ les points forts d’un cours,
- ❌ les irritants,
- 📈 des suggestions d’amélioration,

afin de générer une rétroaction claire destinée au corps professoral.

---

### 🧠 Outils de révision
- Quiz interactifs
- Flashcards
- Jeux-questionnaires
- Révision par matière
- Historique de progression

# ✅ Définition d’une tâche terminée

Une tâche est considérée complétée uniquement si :

- Le code respecte les exigences Jira
- Le code est documenté
- Les tests ont été réalisés
- La Pull Request possède 2 approbations
- Aucun conflit non résolu

---

# 🤝 Contribution

## Processus

1. Créer une branche
2. Développer proprement
3. Documenter le code
4. Ouvrir une PR
5. Obtenir **2 approbations**
6. Fermer le ticket Jira associé

---

# 🧠 Politique IA / Vibe Coding

L’utilisation d’IA est autorisée si :
- le développeur comprend entièrement le code,
- le code est testé rigoureusement,
- la PR contient le label :

```text
vibe-coded
```

> Si tu ne comprends pas ton code, tu ne merges pas ton code.
